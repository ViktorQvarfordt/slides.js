# slides.js Easy to write slides with *Markdown* formatting. Press `F11` to enter full-screen and go to next slide with $\to$.
# Ma$\theta$ematics Superb support for mathematics ($\LaTeX$) via *MathJax*. Both in-line: $\tau = 2\pi$, and display-style: $$\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}.$$
